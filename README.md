# drn_paper
 
Collection of the code used for analysis and plotting of the figures for the Mutlu et al 2024 titled "Topographically organized sensory-motor activity of dorsal raphe and its forebrain innervations modulate resting-state and sensory-evoked forebrain activity and animal behavior"

Each Figure has it's own matlab script and the implemented functions are collected in the "Subfunction" folder.
